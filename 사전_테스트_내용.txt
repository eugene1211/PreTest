언리얼 엔진 최신 버전( 현재 4.27 )을 사용하여 다음과 같은 프로그램을 작성하세요.
# 요약
 1. 유저의 키보드 입력을 받아 다양한 종류의 투사체를 발사하는 캐릭터를 만듭니다.

# 주의사항
 1. 기본으로 제공되는 템플릿 중, 횡스크롤 템플릿( 3D, C++ )을 사용합니다.
 2. 모든 작업물은 C++을 사용합니다.
 3. 비주얼 요소는 고려하지 않습니다. 기본 리소스들을 사용하세요. ( 화살표, 구, 박스 등 )
 4. 작업물은 https://github.com/ 또는 이와 유사한 공개 버전 관리 시스템(bitbucket, gitlab 등)을 이용해 제출하세요.

# 기본 준비 ( Github 기준 )
 1. 언리얼에서 횡스크롤 템플릿 ( 3D, C++ )을 이용해 신규 프로젝트를 생성합니다.
 2. 생성된 기본 템플릿 프로젝트 위치에 Github 공개 repository를 작성합니다. 이 때 UnrealEngine용 .gitignore를 적용합니다.
 3. 커밋 로그는 상세하게 작성합니다.

# 상세 구현
생성된 프로젝트 내에서 다음 내용들을 가능한 범위까지 구현합니다.
( 각 구현들은 동일 캐릭터로부터 확인 가능해야 합니다. )

A. 발사체 
 1. 일반 발사체
  a. Q 키를 눌렀다 떼면 발사됩니다.
  b. 발사체의 외형은 전방 ( X축, 언리얼 기본 값 )을 가리키는 화살표( Arrow Component )를 포함한 액터( Actor )로 만듭니다.
  c. 생성은 캐릭터가 바라보는 방향으로 20 유닛( unit, 언리얼 기본 값 ), 캐릭터 발 밑에서 50 유닛 떨어진 위치에 만듭니다.
  d. 생성 후, 캐릭터가 바라보는 방향으로 직선 운동 합니다. ( 속도는 100 유닛 / 초 )
  e. 3초 후 자동 소멸됩니다.
  f. 캐릭터를 블록 ( Block )하는 모든 종류의 오브젝트와 임의 충돌 시 즉시 소멸합니다.

 2. 충전 발사체
  a. Q 키를 3초 이상 눌렀다 떼면 발사됩니다. ( 1번 발사체는 발사되지 않습니다. )
  b. 1-b 와 동일하되, 화살표의 크기만 3배 확장합니다.
  c. 1-c 와 동일
  d. 1-d 와 동일
  e. 5초 후 자동 소멸합니다.
  f. 1-f 와 동일

 3. 분열 발사체
  a. Q 키를 누른 상태로 1초 이내에 W키를 눌렀을 때 발사됩니다.
  b. 1-b 와 동일한 상태에서, 캐릭터의 전방 벡터 기준 지면에 수평하게 45도( Degree ) 틀어진 화살표를 2개 추가합니다.
  c. 1-c 와 동일
  d. 1-d 와 동일
  e. 3초 후, 1번 발사체를 자신 내부의 화살표 방향들로 각각 생성 발사합니다.
   ( 3개의 1번 발사체를 진행 방향, 양 옆 45도로 발사 )
   ( 자기 자신은 소멸합니다. )
  f. 1-f 와 동일

 4. 반사 발사체
  a. W 키를 눌렀다 떼면 발사됩니다.
  b. 1-b와 동일하되, 화살표의 색깔은 파란색(R=0,G=0,B=1) 입니다.
  c. 1-c와 동일
  d. 1-d와 동일
  e. 2-e와 동일
  f. 캐릭터를 Block 하는 모든 종류의 오브젝트와 임의 충돌 시, 반대 방향으로 1-d 기준 속력으로 이동합니다.

B. UI
 0. 참고
  a. 모든 UI의 로직은 C++에서 작성합니다.
  b. 화면 표기를 위한 Blueprint UserWidget은 로직이 포함된 C++ 클래스를 상속 받습니다.
  c. BP에서 생성된 Widget을 C++에서 사용 시 바인드하여 직접 컨트롤 합니다.
  d. C++ 로직에서 UI 화면 표기 시 Blueprint UserWidget 경로를 하드코딩으로 포함해도 됩니다.

 1. 충전 UI
  a. 충전 발사체를 발사할 수 있는 시간을 표기하기 위한 Progress Bar UI입니다.
  b. 충전 가능한 발사체 키를 누르면 충전 UI가 표시됩니다.
  c. UI의 Progres Bar는 3초 동안 Progress는 0~1로 변화하고, 3초 이상 유지하고 있으면 1 상태를 유지합니다.
  d. 발사체가 발사되면 UI는 사라집니다.

 2. 발사체 카운터 UI
  a. 현재까지 발사된 A-1, A-2, A-3, A-4 발사체의 개수를 각각 카운트 하여 화면에 표시합니다. 

 3. 발사체 카운터 리셋 버튼
  a. Reset 버튼을 누르면 현재까지 카운트 하던 발사체 개수를 전체 초기화 합니다.
